Doctor Booking Website
 Overview

The Doctor Booking Website is an online platform that allows users to search for doctors, book appointments, and manage their healthcare needs seamlessly. Built with React and Tailwind CSS, this application offers a user-friendly interface and responsive design.

 Features

- User Registration and Login: Secure authentication for users to create accounts and log in.
- Doctor Search: Users can search for doctors by specialty, location, and availability.
- Appointment Booking: Users can book appointments directly through the platform.
- Responsive Design: The application is fully responsive and works on mobile and desktop devices.
- User Dashboard: Users can view their booked appointments and manage their profiles.

 Installation

 Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/) (comes with Node.js)

 Steps to Set Up

1. Clone the Repository:
   ```
   git clone https://github.com/Grealishgit/DoctorBookingWebsite.git
   Navigate to the Project Directory:
   ```
2.Navigate to the Project Directory:
```
cd DoctorBookingWebsite
```
3. Install Dependencies:
   ```
   npm install
4.Start the Development Server:

   ```
npm start server
npm run server
```
5.Start the frontend
```
npm run dev
```
6.Start the backend
```
npm run dev
```
7.Start the admin
```
npm run dev
```
Usage
1.Register an Account: Click on the "Register" link to create a new account.
2.Log In: After registration, log in using your credentials.
3.Search for Doctors: Use the search bar to find doctors based on specialty or location.
4.Book an Appointment: Select a doctor and choose an available time slot to book your appointment.
5.Manage Appointments: Visit your dashboard to view and manage your appointments.
